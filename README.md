# Border-radius Previewer

A simple frontend tool for previewing CSS `border-radius` values and generating the corresponding CSS code.

## Features

- live preview of border-radius changes
- input fields for individual corner values
- generated CSS code preview
- simple responsive layout
- built with vanilla JavaScript, HTML and CSS

## Technologies

- HTML
- CSS
- JavaScript

## How to run

Open `index.html` directly in the browser or use a local development server.
